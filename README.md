## ✨ Features

### 🎬 Smooth Scrolling & Animation
- **Locomotive Scroll** integration for buttery-smooth scroll effects
- **GSAP** animations with scroll triggers for dynamic page transitions
- Advanced parallax scrolling with variable scroll speeds

### 🖱️ Interactive Elements
- **Custom Cursor** - Premium animated cursor that scales and responds to interactions
- **Magnetic Navigation** - Navigation elements that attract the cursor using SheryJS
- **Video Integration** - Click-to-play video section with smooth opacity transitions
- **Image Effects** - Gooey blur effects on image galleries using SheryJS

### 🎯 Page Sections
- **Hero Section** - Bold typography with animated entrance transitions
- **Video Showcase** - Embedded video player with interactive controls
- **Project Gallery** - Multi-column image grid with hover-triggered circular info cards
- **About Section** - Agency story with awards showcase (16x Awwwards, 11x FWA, 23x CSSDA, and more)
- **Skills & Industries** - Horizontal scrolling text animation across multiple sectors
- **Footer** - Contact information and social media links

### 📱 Responsive Design
- Optimized for desktop, tablet, and mobile devices
- Adaptive layouts that maintain visual quality across all screen sizes
- Touch-friendly interactive elements for mobile users

### 🎨 Design Highlights
- Dark, premium aesthetic with white typography
- Custom fonts (Plain Light, Silk Serif Italic) for sophisticated typography
- Smooth color transitions and hover effects
- Proper z-index layering for complex animations
